# Corda + Accord Project Template Front End

This repo contains a React front end for the MVP integration between Corda and the Accord Project at [Corda + Accord Project Template](https://github.com/rogersanick/corda-accord-template).

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.
